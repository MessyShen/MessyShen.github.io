选班委

有很多同学报名当班委，然而班委人数有限，为了体现平等民主原则，班委按照以下方案产生(伪代码)。


string 班委候选人[12] = {…};
string 班委[8]; //包括班长团支书

void 选班委(int n){
    if(n == 0) return;
    班委成员讨论
    在班委候选人中选出1个人
    这个人加入班委
    将这个人从班委候选人中删除;
    选班委(n-1)；
}

int main(){
    清空班委;
    班委中加入班长团支书;
    选班委(6);
    return 0;
}

先放一段来自班委的碎碎念
大家翘首以盼、望眼欲穿、魂牵梦萦、日思夜想――咳咳――的班级秋冬游终于来啦~郊游什么的都不重要，吃才是头等大事，露天烧烤什么的真是太迷人了！
本次秋游时间，在南京缠绵的雨后，最终定在了11月5日本周六上午11点，在终于迎来太阳的日子里又等来了烧烤，真是感动到哭。
地点是紫金山脚下的白马公园。公园真小，没什么可逛的，but，谁让吃是头等大事呢。
嗯下面正式进入碎碎念环节――

  1. 如何到达白马公园涅？打的的各位想必不需要我指导了；此外，可以选择搭乘地铁2号线到大行宫站，而后乘2路公交车到太平门；你也可以在紫金东路站乘坐115路到达板仓街·岗子村；当然，计划爬山的各位老司机一定已经有路线规划了，在此不赘言了。
  2. 白马公园门票10元每人，学生证（或一卡通）可打对折，想少花点钱的同学不要忘记带学生证哦~
  3. 本次烧烤预收50元每人的份子钱，多退少补，大家莫忘了将钱转给陈清梅。
  4. 建议自带一些纸巾和湿巾。
  5. 天气转凉，浪里个浪也要保重身体，早睡早起，注意保暖。
  6. 肉要烤熟，调料好好放，烤红薯……嗯，大家好好浪。（流口水.gif）
